# ReadingHistoryList

Owns the refreshable scroll presentation for saved readings. It receives display-ready rows and does not format dates or fetch data.

## Props

- `emptyMessage`: Text shown when there are no rows and loading is false.
- `isLoading`: Drives the refresh control state.
- `onCreateReadingPress`: Called from the empty-state action.
- `onRefresh`: Called by pull-to-refresh.
- `readings`: Display-ready reading rows.
