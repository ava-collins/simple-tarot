## Props

- `error`: Optional generation error text.
- `isGenerating`: Disables the submit button and changes its label.
- `latestReading`: Optional latest reading result to display.
- `onBackPress`: Back navigation callback.
- `onGeneratePress`: Receives the question text.
- `onHistoryPress`: History navigation callback.
