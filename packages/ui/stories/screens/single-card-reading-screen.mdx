Mobile "tap to draw" screen wrapped in `MobileView`. Wraps the `NewReading` organism unmodified and layers in auth, generating, and error presentation around it.

## Responsibilities

- Shows auth loading and signed-out states.
- Blocks double taps while a reading is generating.
- Surfaces an error with a retry action.
- Delegates the tap-to-draw interaction to the `NewReading` organism.
